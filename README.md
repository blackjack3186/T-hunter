# Tautology Hunter 1.1

Инструмент для писателей. Находит тавтологии, повторы, однокоренные слова и слова-сорняки в русском тексте.

Автор: Esquello, Claude

---

## Установка

1. Установи Python 3.8+ с [python.org](https://www.python.org/downloads/)
   - При установке обязательно поставь галку **Add Python to PATH**
2. Скачай файл `tautology_hunter.py`
3. Запусти двойным кликом или через командную строку:

python tautology_hunter.py

4. При первом запуске программа сама установит нужные библиотеки (pyperclip, pymorphy3, python-docx)

---

## Как пользоваться

### Проверка текста на тавтологии

1. Скопируй фрагмент текста из Word или любого редактора (Ctrl+C)
2. Нажми **🔍 Проверить буфер**
3. В окне «Найденные проблемы» появится список:
   - **ПОВТОР** (красный) — одно и то же слово в разных формах рядом
   - **ОДНОКОР** (оранжевый) — однокоренные слова рядом
   - **СОРНЯК** (фиолетовый) — слова из списка нежелательных
4. В окне «Помеченный текст» — превью с цветной подсветкой. Каждая пара повторов выделена своим цветом

### Автоматическая проверка

Включи галку **Автоматически проверять при копировании** — программа будет анализировать каждый скопированный текст без нажатия кнопки.

---

## Режимы пометки

После проверки нажми **📋 Вставить помеченный текст в буфер**. Результат зависит от выбранного режима:

### Пометить звёздочкой*

Проблемные слова оборачиваются в `*звёздочки*`. Вставь текст в любой редактор, нажми Ctrl+F, ищи по `*` — пройдись по всем местам. После правки — Ctrl+H, замени `*` на пустоту.

### Создать DOCX-файл

Программа предложит сохранить новый DOCX-файл. В нём проблемные слова выделены цветом и жирным шрифтом:
- Каждая пара повторов — своим цветом
- Сорняки — фиолетовым с подчёркиванием

Файл автоматически откроется в Word. Можешь копировать и вставлять текст из него в свой документ с рукописью.

---

## Слова-сорняки

Нажми **⚙ Настроить слова-сорняки** — откроется редактор списка. Каждое слово или фраза на отдельной строке.

Примеры сорняков:

    юноша
    побелевшие костяшки
    похолодел
    звенящая тишина
    кивнул
    усмехнулся

Вставляй в этот список слова, которые пишешь неосознанно и хочешь от них избавиться.

- **Сохранить** — список запишется в файл `weeds.txt` рядом со скриптом
- **По умолчанию** — вернуть исходный список
- Файл `weeds.txt` можно редактировать и в блокноте — изменения подхватятся при следующем запуске

## Частотный анализ

Показывает, какие слова встречаются в тексте чаще всего. Помогает найти слова-паразиты, которые автор не замечает.

1. Настрой параметры:
   - **Мин. букв** — слова короче будут пропущены (по умолчанию 4)
   - **Мин. повторов** — порог частоты (по умолчанию 3)
2. Нажми **📊 Анализ частот из файла**
3. Выбери файл (DOCX или TXT)
4. Появится таблица: лемма, пример из текста, количество
5. Что можно сделать:
   - **💾 Сохранить CSV** — экспорт в таблицу (откроется в Excel)
   - **🚫 Добавить в сорняки** — выдели строки в таблице и нажми кнопку, слова добавятся в список сорняков

## Известные особенности

- Анализ основан на лемматизации (pymorphy3) — программа понимает, что «бежал» и «бегу» — одно слово
- Однокоренные определяются грубо (по совпадению начала леммы) — бывают ложные срабатывания. Лучше отсеять лишнее, чем пропустить реальную тавтологию
